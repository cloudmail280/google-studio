# Background music library

Simpan file `.mp3` / `.wav` / `.m4a` di folder ini. Pipeline akan pilih **random** satu track per video.

## Sumber gratis yang direkomendasikan

| Sumber | Link | Catatan lisensi |
|---|---|---|
| **YouTube Audio Library** | https://studio.youtube.com/ → Audio library | Paling aman untuk upload ke YouTube. Filter "No attribution required" |
| **Pixabay Music** | https://pixabay.com/music/ | Pixabay Content License, gratis untuk komersial |
| **Free Music Archive** | https://freemusicarchive.org/ | Campur CC-BY / CC0 — cek per track |
| **Incompetech** (Kevin MacLeod) | https://incompetech.com/music/ | CC-BY, wajib credit di deskripsi |
| **Uppbeat** | https://uppbeat.io/ | Free tier: 10 download/bulan, perlu credit |

## ⚠️ Penting

- **JANGAN** pakai lagu populer / copyrighted. YouTube Content ID akan auto-claim/block video kamu
- Untuk track CC-BY, **wajib cantumkan kredit** di deskripsi video
- Pipeline ini tidak auto-download musik untuk menghindari masalah lisensi. Kamu harus download manual ke folder ini

## Struktur

```
assets/music/
├── upbeat-ambient.mp3
├── dramatic-cinematic.mp3
└── lofi-chill.mp3
```

Makin banyak track di sini, makin variatif audio video kamu. Rekomendasi: 5-10 track dari genre berbeda.
